# Interactive Glass Calendar

This project is an interactive calendar component built using Next.js and React. The goal was to take a static calendar design and turn it into something usable, responsive, and visually engaging.

Instead of keeping it basic, I focused on making the UI feel modern and interactive with a glassmorphism style and smooth user interactions.

---

## What it does

* Lets you select a **start date and end date**
* Clearly highlights the **selected range**
* Includes a **notes section** for the month
* Changes the **visual theme based on the selected month**
* Works smoothly across **desktop and mobile**

---

## Why I built it this way

The challenge required a wall-calendar style layout, so I structured the UI around a **visual panel + calendar grid**.

I also wanted it to feel less like a static component and more like a small product, so I added:

* Dynamic visuals
* Subtle animations
* Clean interaction feedback

---

## Tech used

* Next.js
* React
* TypeScript
* CSS (custom styling, no heavy UI libraries)

---

## Running locally

Clone the repo:

git clone https://github.com/YOUR_USERNAME/interactive-glass-calendar.git

Go into the project folder:

cd interactive-glass-calendar

Install dependencies:

npm install

Start the dev server:

npm run dev

Open in browser:

http://localhost:3000
